# warp
RUSTのターミナル
```
command + p: コマンドパレット一覧
control + r: historyからコマンドを検索・実行
control + @: やりたいことを書けばコマンドを提案してくれる
control + shit + r: workflow表示
command + control + t: テーマ選択
command + ↑ or ↓: ブロック移動
command + shift + c: ブロックのコマンドコピー
```
